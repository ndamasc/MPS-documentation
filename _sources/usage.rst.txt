Setup Application
=================

.. _dependencies:

Dependencies
------------

The following dependencies are required to use MPS-Festo:

- fastapi
- uvicorn[standard]
- pymodbus
- pyModbusTCP
- pyodbc
- rtde_io

.. _installation:

Installation
------------

To install MPS-Festo, clone the repository and install the required dependencies using pip:

.. code-block:: bash

    git clone https://github.com/yourusername/MPS-Festo.git
    cd MPS-Festo
    pip install -r requirements.txt
